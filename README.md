# AI-Ready Smart Colony Digital Twin using OpenDSS

- Overview

This project demonstrates a Smart Colony Distribution Network developed using OpenDSS and Python.

The model includes:

* 11 kV Distribution Grid
* 1000 kVA Distribution Transformer
* Residential Loads
* Commercial Load
* EV Charging Station
* Solar PV Integration
* Battery Energy Storage System (BESS)

- Objectives

* Voltage Profile Analysis
* Distribution Loss Analysis
* EV Impact Assessment
* Solar PV Integration Study
* Battery Storage Evaluation

- Tools Used

* OpenDSS
* Python
* OpenDSSDirect.py
* Matplotlib
* VS Code
